# How to use

Open site -> add numbers -> click button (this will copy the link into your
clipboard) -> paste into obs -> customize in obs with .clock{} css tag

Do note that the css can be scuffed. 
here's my obs css, which I built the rest of the css around
```css
body { background-color: rgba(255,255,255, 0); margin: 0px auto; overflow: hidden; }
.clock {
font-size:600px;
font-family: Arial, Helvetica, sans-serif;
}
```
